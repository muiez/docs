## Two Ways to Use AgentMind

**Python SDK** (Recommended) - 3 lines to get started  
**REST API** - Use from any language

## Python SDK (Fastest)

```python
from agentmind import Memory

# That's it - you're ready
memory = Memory(api_key="am_live_YOUR_API_KEY")

# Store anything
memory.remember("User loves Python")
memory.remember({"config": "production", "debug": False})

# Search naturally
results = memory.recall("What does the user love?")
```

## REST API

```bash
# Base URL
https://api.agent-mind.com/v1

# Auth Header
Authorization: Bearer am_live_YOUR_API_KEY
```

## Quick Example: Store & Search

### Python

```python
# Store
memory_id = memory.remember("User is a Python developer")

# Get by ID
data = memory.get(memory_id)

# Search
results = memory.recall("What language does the user know?")
```

### REST API

```bash
# Store
curl -X POST https://api.agent-mind.com/v1/memories \
  -H "Authorization: Bearer am_live_YOUR_API_KEY" \
  -d '{"content": "User is a Python developer"}'

# Search  
curl -X POST https://api.agent-mind.com/v1/memories/search \
  -H "Authorization: Bearer am_live_YOUR_API_KEY" \
  -d '{"query": "What language does the user know?"}'
```
