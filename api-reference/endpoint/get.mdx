<Note>
  **HTTP Method:** `GET`  
  **Endpoint:** `https://api.agent-mind.com/v1/memories/{id}`
</Note>

## Overview

Retrieve a specific memory entry by its unique identifier. This is the fastest way to access memories when you know the exact ID.

## Request

<ParamField path="id" type="string" required>
  The unique identifier of the memory to retrieve
</ParamField>

<ParamField query="include_metadata" type="boolean" default="false">
  Include metadata in the response
</ParamField>

<ParamField query="include_embedding" type="boolean" default="false">
  Include the vector embedding in the response
</ParamField>

## Response

<ResponseField name="id" type="string">
  The unique identifier of the memory
</ResponseField>

<ResponseField name="content" type="any">
  The stored memory content (string, object, or array)
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp of when the memory was created
</ResponseField>

<ResponseField name="metadata" type="object">
  Additional metadata (only if include_metadata=true)
</ResponseField>

<ResponseField name="embedding" type="array">
  Vector embedding for semantic search (only if include_embedding=true)
</ResponseField>

## Examples

### Basic Retrieval

<CodeGroup>

```python Python
from agentmind import Memory

memory = Memory(api_key="YOUR_API_KEY")

# Get memory by ID
content = memory.get("mem_abc123")
```

```bash cURL
curl https://api.agent-mind.com/v1/memories/mem_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.agent-mind.com/v1/memories/mem_abc123', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
```

</CodeGroup>

### Get with Metadata

<CodeGroup>

```python Python
# Get with metadata included
data = memory.get("mem_abc123", include_metadata=True)
```

```bash cURL
curl "https://api.agent-mind.com/v1/memories/mem_abc123?include_metadata=true" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.agent-mind.com/v1/memories/mem_abc123?include_metadata=true',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
```

</CodeGroup>

## Performance Notes

- Direct ID lookup is O(1) and typically returns in less than 10ms
- Including embeddings adds approximately 2-5ms to response time
- Memory content is returned exactly as stored (preserves types)
