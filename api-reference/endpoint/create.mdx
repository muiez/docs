---
title: 'Store Memory'
api: 'POST /memories'
description: 'Store new information in agent memory'
---

<Note>
  **HTTP Method:** `POST`  
  **Endpoint:** `https://api.agent-mind.com/v1/memories`
</Note>

## Overview

Store a new memory entry in your agent's memory bank. The content can be any JSON-serializable data including strings, dictionaries, lists, or complex nested structures.

## Request

<ParamField body="content" type="any" required>
  The data to store in memory. Can be a string, dictionary, list, or any JSON-serializable object.
</ParamField>

<ParamField body="id" type="string">
  Custom memory ID. If not provided, a unique ID will be auto-generated. Must be alphanumeric with underscores only.
</ParamField>

<ParamField body="metadata" type="object">
  Additional metadata to attach to the memory for filtering and organization.
</ParamField>

<ParamField body="user_id" type="string">
  Associate memory with a specific user or conversation.
</ParamField>

<ParamField body="session_id" type="string">
  Group memories by session for context management.
</ParamField>

<ParamField body="ttl" type="integer">
  Time-to-live in seconds. Memory will auto-expire after this duration.
</ParamField>

## Response

<ResponseField name="id" type="string">
  The unique identifier for the stored memory
</ResponseField>

<ResponseField name="status" type="string">
  Status of the operation ("success")
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp of when the memory was created
</ResponseField>

## Examples

### Store User Preferences

<CodeGroup>

```python Python
from agentmind import Memory

memory = Memory(api_key="am_live_YOUR_API_KEY")

memory.remember(
    "Prefers email communication, no calls before 10am PST",
    metadata={
        "user_id": "user_123",
        "category": "preferences",
        "type": "communication"
    }
)
```

```bash cURL
curl -X POST https://api.agent-mind.com/v1/memories \
  -H "Authorization: Bearer am_live_YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Prefers email communication, no calls before 10am PST",
    "metadata": {
      "user_id": "user_123",
      "category": "preferences",
      "type": "communication"
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.agent-mind.com/v1/memories', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer am_live_YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    content: "Prefers email communication, no calls before 10am PST",
    metadata: {
      user_id: "user_123",
      category: "preferences",
      type: "communication"
    }
  })
});

const data = await response.json();
```

</CodeGroup>

## Error Responses

### 400 Bad Request

```json
{
  "error": {
    "code": "invalid_id",
    "message": "Memory ID contains invalid characters",
    "details": {
      "field": "id",
      "value": "my-memory-123",
      "reason": "Only alphanumeric and underscores allowed"
    }
  }
}
```

### 401 Unauthorized

```json
{
  "error": {
    "code": "invalid_api_key",
    "message": "The provided API key is invalid or expired"
  }
}
```

### 413 Payload Too Large

```json
{
  "error": {
    "code": "content_too_large",
    "message": "Memory content exceeds maximum size of 1MB"
  }
}
```
