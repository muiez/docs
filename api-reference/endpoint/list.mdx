title: 'List Memories'

description: 'List memories owned by authenticated user'
authMethod: 'bearer'
---

## Overview

List all memories stored, with optional filtering by user_id. This endpoint supports pagination for large memory collections.

## Query Parameters

<ParamField query="limit" type="integer" default="50">
  Maximum number of memories to return per page. Default is 50.
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of memories to skip for pagination. Default is 0.
</ParamField>

<ParamField query="user_id" type="string">
  Optional filter to list memories for a specific user only.
</ParamField>

## Response

Returns a paginated list of memories with their metadata and content.

## Examples

### List All Memories

<CodeGroup>

```python Python
import requests

url = "https://api.agent-mind.com/api/v1/memories/list"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}
params = {
    "limit": 50,
    "offset": 0
}

response = requests.get(url, headers=headers, params=params)
memories = response.json()
```

```bash cURL
curl -X GET "https://api.agent-mind.com/api/v1/memories/list?limit=50&offset=0" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const url = new URL('https://api.agent-mind.com/api/v1/memories/list');
url.searchParams.append('limit', '50');
url.searchParams.append('offset', '0');

const response = await fetch(url, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const memories = await response.json();
```

</CodeGroup>
