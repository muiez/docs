---
title: 'Update Memory'
api: 'PUT /api/v1/memories/update/{memory_id}'
description: "Update a memory's content and metadata"
authMethod: 'bearer'
---

## Overview

Update an existing memory's content and metadata. This endpoint allows you to modify any aspect of a stored memory.

## Path Parameters

<ParamField path="memory_id" type="string" required>
  The unique identifier of the memory to update.
</ParamField>

## Request Body

<ParamField body="content" type="string">
  Updated content for the memory.
</ParamField>

<ParamField body="category" type="string">
  Category classification for the memory.
</ParamField>

<ParamField body="tags" type="array">
  Array of tags to associate with the memory.
</ParamField>

<ParamField body="importance" type="number">
  Importance score (0-1) for memory prioritization.
</ParamField>

<ParamField body="confidence" type="number">
  Confidence score (0-1) for the memory's accuracy.
</ParamField>

<ParamField body="source" type="string">
  Source of the memory information.
</ParamField>

<ParamField body="user_id" type="string">
  Updated user association for the memory.
</ParamField>

<ParamField body="session_id" type="string">
  Updated session association for the memory.
</ParamField>

<ParamField body="metadata" type="object">
  Additional metadata to attach or update.
</ParamField>

## Response

Returns the updated memory object with all current values.

## Examples

### Update Memory Content

<CodeGroup>

```python Python
import requests

memory_id = "mem_abc123"
url = f"https://api.agent-mind.com/api/v1/memories/update/{memory_id}"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "content": "User strongly prefers email over phone calls",
    "importance": 0.9
}

response = requests.put(url, json=data, headers=headers)
updated_memory = response.json()
```

```bash cURL
curl -X PUT https://api.agent-mind.com/api/v1/memories/update/mem_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type": "application/json" \
  -d '{
    "content": "User strongly prefers email over phone calls",
    "importance": 0.9
  }'
```

```javascript JavaScript
const memoryId = "mem_abc123";
const response = await fetch(`https://api.agent-mind.com/api/v1/memories/update/${memoryId}`, {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    content: "User strongly prefers email over phone calls",
    importance: 0.9
  })
});

const updatedMemory = await response.json();
```

</CodeGroup>
